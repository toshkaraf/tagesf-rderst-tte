# Решение проблем с запуском сервера

## Быстрый запуск

**Дважды кликните на файл:** `ЗАПУСТИТЬ_ЗДЕСЬ.bat`

Этот файл автоматически:
- Проверит наличие Node.js и npm
- Установит зависимости (если нужно)
- Запустит сервер

## Если сервер все равно не запускается

### 1. Проверьте установку Node.js

Откройте командную строку и введите:
```cmd
node --version
npm --version
```

Если команды не работают, установите Node.js:
- Скачайте с сайта: https://nodejs.org/
- Установите версию LTS (рекомендуется)
- Перезагрузите компьютер после установки

### 2. Установите зависимости вручную

Откройте командную строку в папке проекта:
```cmd
cd c:\dev\projects\Tagesförderstätte
npm install
```

### 3. Запустите сервер вручную

```cmd
npm run dev
```

### 4. Проверьте, не занят ли порт 3000

Если порт занят, измените порт в файле `vite.config.ts`:
```typescript
server: {
  port: 3001,  // Измените на 3001 или другой свободный порт
  ...
}
```

### 5. Посмотрите ошибки

В окне, где запускается сервер, будут видны все ошибки. 
Скопируйте текст ошибки и проверьте:
- Нет ли ошибок в коде
- Установлены ли все зависимости
- Правильно ли настроен проект

## Что делать, если ничего не помогает

1. Удалите папку `node_modules`
2. Удалите файл `package-lock.json` (если есть)
3. Запустите `npm install` заново
4. Попробуйте запустить сервер снова

## Контакты для помощи

Если проблема остается, опишите:
- Какую ошибку вы видите
- На каком этапе возникает проблема
- Версию Node.js (node --version)
