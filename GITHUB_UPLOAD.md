# Загрузка проекта в GitHub

## Инструкция по загрузке

### Вариант 1: Создание нового репозитория на GitHub

1. Перейдите на https://github.com
2. Нажмите кнопку "New" или "+" → "New repository"
3. Назовите репозиторий, например: `Tagesförderstätte` или `tagesforderstatte`
4. НЕ создавайте README, .gitignore или лицензию (они уже есть)
5. Скопируйте URL репозитория (например: `https://github.com/ваш-username/tagesforderstatte.git`)

### Вариант 2: Использование существующего репозитория

Если у вас уже есть репозиторий, скопируйте его URL.

## Команды для загрузки

После того как у вас есть URL репозитория, выполните:

```bash
cd c:\dev\projects\Tagesförderstätte

# Добавить удаленный репозиторий
git remote add origin https://github.com/ваш-username/название-репозитория.git

# Переименовать основную ветку в main (если нужно)
git branch -M main

# Отправить код в GitHub
git push -u origin main
```
