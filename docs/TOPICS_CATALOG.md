# Каталог тем для интерактивных занятий (100 тем)

## Принцип отбора тем

Каждая тема построена на **нетривиальных фактах** для образованного человека без специального образования:
- Неожиданные детали известных событий
- Экономические и политические аспекты исторических явлений
- Реальность vs романтизированные представления
- Цифры и факты, которые удивляют
- 40 слайдов и 30 заключительных вопросов

## Структура каждой темы

Каждая тема рассчитана на 60-90 минут занятия и включает:

1. **40 слайдов с нетривиальными фактами** (45-60 мин)
   - Экономические аспекты
   - Статистика и цифры
   - Развенчание мифов
   - Неожиданные детали

2. **10 блоков викторин по 3 вопроса** (20-30 мин)
   - После каждых 4-5 слайдов
   - Проверка запоминания конкретных фактов

3. **Итоговая викторина** (10-15 мин)
   - Все 30 вопросов в случайном порядке
   - Подсчёт результатов

## Принципы создания контента

✅ **Нетривиальность**: факты, которые удивляют
✅ **Конкретика**: цифры, суммы, статистика
✅ **Реализм**: без романтизации истории
✅ **Сложность**: для образованной аудитории, но без специальных знаний
✅ **Немецкий B2**: понятный, но не упрощённый язык
