# Структура проекта Tagesförderstätte

## Обзор

Проект создан для проведения интерактивных образовательных занятий с клиентами. Поддерживает различные форматы контента: презентации, викторины, видео, аудио, изображения.

## Архитектура

### Frontend (React + TypeScript)

- **React 18** - современный UI фреймворк
- **TypeScript** - типобезопасность
- **Vite** - быстрая сборка
- **React Router** - навигация

### Структура данных

#### Session (Занятие)
Основная единица контента. Каждое занятие включает:
- Метаданные (название, описание, категория, тип)
- Структуру занятия (приветствие, основная часть, интерактив, завершение)
- Слайды с контентом
- Викторины
- Медиа материалы

#### Slide (Слайд)
Элемент презентации:
- Заголовок и текст
- Медиа (изображения, видео, аудио)
- Вопросы для интерактивности
- Длительность

#### MediaItem (Медиа элемент)
- Тип (image/video/audio)
- URL
- Описание

#### Question (Вопрос)
- Тип (yes-no/multiple-choice/open)
- Варианты ответов
- Правильный ответ
- Объяснение

## Компоненты

### Pages (Страницы)

1. **HomePage** - главная страница со списком занятий
   - Отображение по категориям
   - Фильтрация по типу
   - Поиск

2. **SessionPage** - страница занятия
   - Навигация по слайдам
   - Просмотр медиа
   - Интерактивные вопросы
   - Викторина

### Components (Компоненты)

1. **LargeButton** - крупные кнопки для планшета
   - Минимальная высота 80px
   - Крупный шрифт (24px)
   - Удобные для нажатия пальцем

2. **SlideViewer** - просмотр слайдов
   - Отображение текста
   - Медиа контент
   - Вопросы

3. **QuizViewer** - викторина
   - Вопросы разных типов
   - Подсчет баллов
   - Объяснения

4. **MediaViewer** - медиа контент
   - Изображения
   - Видео с контролами
   - Аудио плеер

## Типы занятий

### 1. Presentation (Презентация)
Классическая презентация с рассказом и слайдами

### 2. Quiz (Викторина)
Интерактивная викторина с вопросами

### 3. Virtual Tour (Виртуальное путешествие)
Экскурсии по городам, музеям, историческим местам

### 4. Biography (Биография)
История жизни известных личностей

История искусства, музыки, литературы

Научно-популярные темы и открытия

### 7. Language (Языковая тема)
Этимология, сравнение языков

### 8. Discussion (Тематическая дискуссия)
Обсуждение проблемных вопросов

## Медиа контент

### Хранение
Медиа файлы хранятся в `public/media/` с организацией по темам:
```
public/media/
├── rome/
│   ├── augustus.jpg
│   └── colosseum.jpg
├── berlin/
│   └── 1920s.jpg
└── music/
    └── baroque.mp3
```

### Форматы
- **Изображения**: JPG, PNG, WebP
- **Видео**: MP4, WebM
- **Аудио**: MP3, OGG, WAV

## UX для планшета

### Крупные элементы
- Кнопки: минимум 80px высотой
- Шрифт: 24px для кнопок, 22px для текста
- Отступы: 24px между элементами
- Карточки: padding 32px

### Навигация
- Крупные кнопки "Назад" и "Далее"
- Прогресс-бар
- Визуальная обратная связь

### Интерактивность
- Вопросы "Да/Нет" с крупными кнопками
- Множественный выбор с четкими вариантами
- Мгновенная обратная связь

## Расширение функционала

### Добавление нового типа занятия
1. Добавить тип в `SessionType`
2. Добавить иконку в `typeIcons`
3. Добавить метку в `typeLabels`
4. Создать компонент для отображения (если нужен)

### Добавление медиа
1. Загрузить файл в `public/media/`
2. Добавить ссылку в занятие
3. Компонент `MediaViewer` автоматически определит тип

### Создание нового занятия
1. Создать объект `Session` в `sessions.ts`
2. Добавить слайды, медиа, викторину
3. Занятие появится на главной странице
