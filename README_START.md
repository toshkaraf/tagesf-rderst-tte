# Инструкция по запуску сервера

## Быстрый запуск

1. Откройте PowerShell или командную строку в папке проекта

2. Убедитесь, что зависимости установлены:
   ```powershell
   npm install
   ```

3. Запустите сервер:
   ```powershell
   npm run dev
   ```

4. Откройте браузер и перейдите на: **http://localhost:3000**

## Если сервер не запускается

### Проверьте:
- Установлен ли Node.js (версия 18 или выше)
- Установлены ли зависимости (`npm install`)
- Не занят ли порт 3000 другим приложением

### Решение проблем:

1. **Остановите все процессы Node.js:**
   ```powershell
   Get-Process -Name node | Stop-Process -Force
   ```

2. **Проверьте, установлены ли зависимости:**
   ```powershell
   Test-Path node_modules
   ```
   Если `False`, выполните `npm install`

3. **Попробуйте запустить с другим портом:**
   Измените в `vite.config.ts` порт на другой (например, 3001)

4. **Проверьте логи ошибок:**
   В окне, где запускается сервер, будут видны все ошибки

## Альтернативный способ запуска

Запустите файл `start-server.ps1`:
```powershell
powershell -ExecutionPolicy Bypass -File start-server.ps1
```
