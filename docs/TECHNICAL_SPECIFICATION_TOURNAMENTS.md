# Техническое задание

## Общая информация

**Проект:** Интерактивная образовательная презентация о рыцарских турнирах  
**Язык:** Русский  
**Целевая аудитория:** Взрослые с хорошим общим средним образованием, но без специального образования по теме  
**Тип:** Интерактивная сессия с текстовыми слайдами, вопросами и финальной викториной

---

## Основные требования к контенту

### 1. Количество и структура слайдов

- **Общее количество текстовых слайдов:** 40 слайдов прмерно
- **Структура:**
  - Текстовые слайды с информацией (40 штук)
  - Вопросы викторины после логических групп слайдов по пройденным слайдам (по одному вопросу на отдельном слайде)
  - Финальная викторина со всеми вопросами в случайном порядке

### 2. Требования к текстовым слайдам

#### 2.1. Объем текста
- **На каждом слайде:** 2-3 предложения
  - Либо 3 коротких предложения
  - Либо 2 более длинных предложения (но не чрезмерно длинных)

#### 2.2. Качество контента
- **Уровень сложности:** Информация должна быть не совсем банальной
- **Целевая аудитория:** Рассчитана на человека с хорошим полным средним образованием, но без специального образования по теме
- **Стиль:** Компактный, без лишних слов
- **Повторения:** Строго запрещены - информация в слайдах не должна повторяться

#### 2.3. Интересность
- Включить больше интересных фактов
- Избегать тривиальной информации
- Предоставлять неочевидные детали, которые расширяют общие знания

#### 2.4. Медиа-контент
- **Обязательное требование:** Все текстовые слайды должны сопровождаться изображением
- Каждый слайд должен иметь соответствующую картинку

### 3. Требования к вопросам

#### 3.1. Расположение вопросов
- Вопросы размещаются **после нескольких логически объединенных слайдов**
- Каждый вопрос размещается на **отдельном слайде** (не группировать несколько вопросов на одном слайде)
- Ответы должны располагаться **друг под другом** (вертикальный список)

#### 3.2. Типы вопросов
- Вопросы с вариантами ответов (multiple-choice)

#### 3.3. Качество вопросов
- Вопросы должны быть **более интересными и менее банальными**
- Избегать тривиальных вопросов
- Вопросы должны проверять понимание, а не только память

#### 3.4. Отображение правильных ответов
- **Запрещено:** Выделять правильный ответ сразу в вопросах
- Правильный ответ показывается только после выбора пользователя

#### 3.5. Вводные тексты
- **Запрещено:** Вводные тексты в викторинах (например, "Проверим, что вы запомнили о первых турнирах! ...")
- Вопросы должны начинаться сразу без вводных фраз

### 4. Финальная викторина

#### 4.1. Структура
- Размещается **после всех слайдов**
- Содержит **все вопросы**, которые были заданы между текстовыми слайдами
- Вопросы представлены **в случайном порядке** (перемешаны)

#### 4.2. Результаты
- В конце финальной викторины показывается **результат** (количество правильных ответов)
