# Требования к темам, слайдам и текстам

## 📋 Общие требования к темам

### Структура темы

Каждая тема должна содержать:

1. **Метаданные**:
   - `id`: уникальный идентификатор (латиница, дефисы, например: `mongol-empire`)
   - `title`: название темы (русский)
   - `description`: краткое описание (2-3 предложения)
   - `type`: тип занятия (`discussion`, `presentation`, `biography`, и т.д.)
   - `category`: категория (`История`, `География`, `Наука`, и т.д.)
   - `duration`: общая длительность в минутах (обычно 60-90)
   - `difficulty`: уровень сложности (`easy`, `medium`, `hard`)

2. **Структура занятия**:
   ```typescript
   structure: {
     greeting: 'Приветствие участникам',
     mainContent: 'Описание основной части',
     interactive: 'Описание интерактивной части',
     additional: 'Дополнительная информация',
     closing: 'Заключительные слова'
   }
   ```

3. **Слайды**: ровно **40 слайдов** с контентом
4. **Викторины**: **10 промежуточных блоков** по 3 вопроса каждый
5. **Итоговая викторина**: **30 вопросов** (все вопросы из промежуточных блоков)
6. **Подбор изображений**: для каждой темы **обязательно** создаётся файл **`IMAGE_LIST.html`** в папке `public/media/theme-id/`. В нём — интерактивный список всех 40 изображений с описаниями, поисковыми запросами и ссылками на Google (картинки), Wikimedia, Pixabay для удобного подбора картинок; чекбоксы и сохранение прогресса в браузере.

## 📝 Требования к слайдам

### Количество и распределение

- **Всего слайдов**: 40
- **Распределение**: 
  - 30 информационных слайдов (с контентом)
  - 10 викторин-блоков (после каждых 4 слайдов)
- **Порядок**: информационный слайд → информационный → информационный → информационный → викторина → повтор

### Структура слайда

```typescript
{
  id: 'уникальный-id',           // Например: 'm1', 'm2', 'q1'
  title: 'Заголовок слайда',     // 3-8 слов, конкретный
  content: 'Текст слайда',       // 2-3 предложения (см. требования ниже)
  duration: 3,                   // Длительность в минутах (обычно 3)
  media: [                       // Опционально: изображения, видео, аудио
    {
      type: 'image' | 'video' | 'audio',
      url: '/media/theme/filename.jpg',
      title: 'Название медиа'
    }
  ]
}
```

### Именование ID слайдов

- **Информационные слайды**: префикс + номер
  - Примеры: `m1`, `m2`, `m3` (для монголов)
  - Примеры: `o1`, `o2`, `o3` (для олимпийских игр)
  - Примеры: `a1`, `a2`, `a3` (для Александра)
  
- **Викторины**: `q1`, `q2`, `q3`, ..., `q10`
  - q1 после слайдов 1-4
  - q2 после слайдов 5-8
  - q3 после слайдов 9-12
  - и т.д.

## ✍️ Требования к текстам слайдов

### Длина и структура

- **Объем**: 2-3 предложения (не более 150 слов)
- **Структура**: 
  - Первое предложение: основной факт
  - Второе предложение: детали, цифры, контекст
  - Третье предложение (опционально): вывод или связь с темой

### Стиль и содержание

✅ **Обязательно**:
- **Конкретика**: цифры, суммы, даты, статистика — **умеренно**, только когда это действительно необходимо для понимания
  - Пример: "Монголы проходили 100-150 км в день"
  - Цены и суммы: не перегружать; если приводим древние/исторические цены — обязательно даём **сопоставление** (сегодняшние деньги или эквивалент: сколько обычный работник тогда должен был работать, чтобы это заработать и т.п.; максимально корректно)
  
- **Нетривиальность**: факты, которые удивляют
  - Пример: "Реальная армия Чингисхана насчитывала 20,000-30,000 всадников, а не миллионы"
  - Пример: "Монголы не тащили обозы — все необходимое везли на лошадях"
  
- **Реализм**: без романтизации, показывать реальность
  - Пример: "Олимпиада была бизнесом с первого дня, прикрытым религией"
  - Пример: "Монголы уничтожали только тех, кто сопротивлялся — экономика важнее мести"

- **Экономический аспект**: где возможно, упоминать финансы
  - Пример: "Один крупный город давал 100,000-500,000 талантов"
  - Пример: "Ямская почта стоила 50,000 талантов в год, но приносила 200,000"

❌ **Запрещено**:
- Общие фразы без конкретики ("было важно", "играло роль")
- Романтизация ("великие герои", "благородные рыцари")
- Слишком сложные термины без объяснения
- Длинные предложения (более 25 слов)
- **Повторения**: информация на слайдах не должна повторяться. Каждый слайд должен содержать **уникальный** факт или угол зрения, не дублирующий другие слайды.

### Развлекательные слайды: юмор и анекдоты

- **Частота**: примерно через каждые **7–10 слайдов** включать один слайд с развлекательным контентом.
- **Типы юмора**:
  - **Умные анекдоты** по теме — остроумные истории, шутки, парадоксы, связанные с темой.
  - **Исторические анекдоты и байки** — реальные или полулегендарные случаи, курьёзы, забавные детали.
- **Содержание**: развлекательные истории без обязательной «учебной» нагрузки.
- **Длина**: в идеале 3–4 предложения; если не укладывается — допустим чуть более длинный текст, но **максимально кратко**.

### Любопытный факт

- **Назначение**: слайды категории **«Любопытный факт»** содержат одну очень необычную, частную деталь по теме.
- **Что это**: то, о чём редко услышишь в обычном рассказе — парадоксальные цифры, курьёзные детали, неожиданные связи, малоизвестные факты.
- **Частота**: 1–2 слайда на тему (по усмотрению).
- **Примеры**: «Сколько энергии съедает обучение одной большой модели ИИ», «Кто первым использовал термин „искусственный интеллект“ и при каких обстоятельствах», «Почему первый „умный“ холодильник провалился» и т.п.

### Примеры хороших текстов

**Пример 1** (Монголы):
```
Монгольский всадник вел 3-5 лошадей. Одна для езды, остальные — запас еды. 
В походе монголы пили кобылье молоко (кумыс) и ели конину. Одна лошадь = 200 кг мяса. 
Это решало проблему провианта в степи.
```

**Пример 2** (Олимпийские игры):
```
В 388 году до н.э. спартанец Евпол подкупил трех боксеров, чтобы они проиграли ему. 
Скандал раскрылся. Элланодики оштрафовали всех четверых на 6 талантов каждого. 
На эти деньги поставили шесть бронзовых статуй Зевса у входа в стадион — Заны. 
Это были монументы позора.
```

**Пример 3** (Александр):
```
Александр потратил 180 талантов на похороны Гефестиона — больше, чем годовой доход Афин. 
Это было политическое заявление: показать богатство и власть. 
Но казна была пуста, и через год после смерти Александра империя обанкротилась.
```

## 🎯 Требования к викторинам

### Промежуточные викторины (10 блоков)

**Структура**:
- **ID**: `q1`, `q2`, ..., `q10`
- **Заголовок**: "Викторина"
- **Количество вопросов**: ровно **3 вопроса** в каждом блоке
- **Расположение**: после каждых 4 информационных слайдов

**Типы вопросов**:
- Только `multiple-choice` (выбор из вариантов)
- **Варианты ответов**: 4 варианта
- **Правильный ответ**: индекс (0, 1, 2, или 3)
- **Объяснение**: обязательное, 1-2 предложения

**Пример**:
```typescript
{
  id: 'q-myth-1',
  text: 'Сколько всадников было в армии Чингисхана?',
  type: 'multiple-choice',
  options: ['100,000', '500,000', '20,000-30,000', '1,000,000'],
  correctAnswer: 2,
  explanation: 'Реальная армия Чингисхана насчитывала 20,000-30,000 всадников, а не миллионы.'
}
```

### Итоговая викторина

- **Количество вопросов**: ровно **30**
- **Источник**: все вопросы из промежуточных викторин (q1-q10, по 3 вопроса = 30)
- **Порядок**: случайный (при отображении)
- **ID вопросов**: `final-q1`, `final-q2`, ..., `final-q30`

## 🖼️ Требования к медиа

### Изображения

- **Количество**: по 1 изображению на каждый информационный слайд (40 изображений)
- **Формат**: JPG или PNG
- **Разрешение**: минимум 1200x800 px
- **Размер файла**: не более 500 KB
- **Лицензия**: свободная (Public Domain, Creative Commons, Pixabay, Unsplash)
- **Источники**: Wikimedia Commons, Pixabay, Unsplash, Pexels
- **Именование**: описательное, на английском, например: `myth-millions.jpg`, `horses-food.jpg`

### Видео

- **Формат**: YouTube ссылки (полный URL)
- **Использование**: опционально, для ключевых моментов
- **Примеры**: битвы, важные события, обобщающие видео
- **Тип**: `{ type: 'video', url: 'https://www.youtube.com/watch?v=...', title: 'Название' }`

### Аудио

- **Использование**: редко, только если необходимо
- **Формат**: MP3, OGG, WAV
- **Размер**: разумный (не более 5 MB)

### Организация медиа

- **Папка**: `public/media/theme-id/`
- **Файлы**:
  - `.gitkeep` — для сохранения папки в Git
  - `README.md` — описание темы и структуры
  - `IMAGE_LIST.md` — список всех изображений с поисковыми запросами (рекомендуется)
  - **`IMAGE_LIST.html`** — **обязателен**. Интерактивная HTML-страница для подбора всех картинок к теме:
    - 40 пунктов (по одному на каждый информационный слайд) с именем файла и кратким описанием;
    - для каждого пункта — поисковые запросы и ссылки на **Google (картинки)**, **Wikimedia Commons**, **Pixabay** (и при необходимости Unsplash), чтобы быстро находить и скачивать изображения;
    - чекбоксы «Загружено» и сохранение прогресса в браузере (localStorage);
    - инструкции по формату, разрешению и лицензии.  
    Открывается в браузере; по ссылкам удобно искать картинки в Google и других источниках, сохранять под нужным именем в `public/media/theme-id/`.

## 🌍 Требования к переводам

### Двуязычность

Каждая тема должна иметь:
- **Русскую версию**: в `src/data/sessions.ts`
- **Немецкую версию**: в `src/i18n/sessionContentTranslations.ts` и `sessionTranslations.ts`

### Немецкий язык (B2 уровень)

**Требования**:
- Понятный для уровня B2 (не упрощённый, но доступный)
- Использование стандартной лексики
- Избегание слишком сложных терминов без объяснения
- Сохранение стиля и тона оригинала

**Что переводится**:
1. **Метаданные** (`sessionTranslations.ts`):
   - `title`, `description`, `category`
   - `structure` (greeting, mainContent, interactive, additional, closing)

2. **Контент** (`sessionContentTranslations.ts`):
   - Все 40 слайдов (title, content)
   - Все 10 промежуточных викторин (вопросы, варианты, объяснения)
   - Итоговая викторина (30 вопросов)

**Пример перевода**:
```typescript
// Русский
{ id: 'm1', title: 'Миф о миллионах', content: 'Европейские хронисты писали о "миллионах" монголов...' }

// Немецкий
{ id: 'm1', title: 'Mythos über Millionen', content: 'Europäische Chronisten schrieben von "Millionen" Mongolen...' }
```

## 📚 Примеры хороших тем

### ✅ Хорошие примеры

1. **"Монгольская империя: мифы о численности и экономика завоеваний"**
   - Конкретные цифры (20,000-30,000 всадников)
   - Экономические аспекты (налоги vs грабеж)
   - Развенчание мифов
   - 40 слайдов, 10 викторин, итоговая викторина

2. **"Олимпийские игры: коррупция и допинг в античности"**
   - Конкретные случаи коррупции (388 до н.э., 332 до н.э.)
   - Экономические детали (100 талантов, 30 талантов)
   - Реалистичный взгляд на спорт
   - Полная структура

3. **"Александр Македонский: цена империи"**
   - Финансовые детали (180 талантов на похороны)
   - Логистика (50,000 армия)
   - Экономический коллапс после смерти

### ❌ Плохие примеры (чего избегать)

1. **Слишком общие темы**:
   - ❌ "История Древнего Рима" (слишком широко)
   - ✅ "Повседневная жизнь в Риме: инсулы и нищета" (конкретно)

2. **Без конкретики**:
   - ❌ "Монголы были сильными" (нет цифр)
   - ✅ "Монголы проходили 100-150 км в день" (конкретика)

3. **Романтизация**:
   - ❌ "Благородные рыцари сражались за честь"
   - ✅ "Рыцари зарабатывали на выкупах пленных"
