# Инструкция для создания текстов к слайдам

> **Как использовать:** Скопируй этот файл или его содержимое и передай другому ИИ вместе с названием темы из каталога. Можно начать с промпта: *«Создай тексты к слайдам для образовательной программы по теме [название]. Следуй инструкциям из приложенного документа.»*

---

**Цель:** Создать тексты для интерактивной образовательной программы (40 слайдов: 30 информационных + 10 викторин + итоговая викторина из 30 вопросов).

**Аудитория:** Образованный взрослый человек с высшим образованием, без специального образования по теме. Программа используется для занятий с человеком с ограниченными возможностями (парализован, плохо говорит), который может нажимать кнопки на планшете.

---

## Структура программы

### Объём
- **30 информационных слайдов** с контентом
- **10 слайдов-викторин** (после каждых 3–4 информационных слайдов)
- **Порядок:** 4 информационных слайда → викторина (3 вопроса) → 4 информационных → викторина → … и так до 10 блоков
- **Итоговая викторина:** 30 вопросов (все вопросы из промежуточных викторин)

### Идентификаторы слайдов
- **Информационные:** префикс + номер (например: `s1`, `s2`, `s3` для темы «Спарта»; `r1`, `r2` для Рима; `p1`, `p2` для философии)
- **Викторины:** `q1`, `q2`, … `q10`

---

## Требования к текстам информационных слайдов

### Длина и структура
- **Объём:** 2–3 предложения (не более 150 слов на слайд)
- **Структура:**
  - Первое предложение: основной факт
  - Второе: детали, цифры, контекст
  - Третье (опционально): вывод или связь с темой

### Стиль и содержание

**✅ Обязательно:**
- **Конкретика:** цифры, даты, суммы — **умеренно**, только когда это необходимо для понимания
- **Цены и суммы:** если приводим древние/исторические цены — обязательно даём **сопоставление** (сегодняшние деньги или эквивалент: сколько обычный работник тогда должен был работать, чтобы это заработать)
- **Нетривиальность:** факты, которые удивляют образованного человека
- **Реализм:** без романтизации, показывать реальность
- **Экономический аспект:** где возможно, упоминать финансы, затраты, выгоду

**❌ Запрещено:**
- Общие фразы без конкретики («было важно», «играло роль»)
- Романтизация («великие герои», «благородные рыцари»)
- Слишком сложные термины без объяснения
- Длинные предложения (более 25 слов)
- **Повторения:** каждый слайд должен содержать **уникальный** факт; один и тот же факт не повторяется на других слайдах

### Развлекательные слайды (анекдоты)
- **Частота:** примерно через каждые **7–10 слайдов** — один слайд с историческим анекдотом или занимательной историей
- **Заголовок:** начинается с «Юмор: » (например: «Юмор: Диоген и Александр»)
- **Содержание:** развлекательная история по теме, без обязательной «учебной» нагрузки
- **Длина:** 3–4 предложения; если не укладывается — допустим чуть длиннее, но кратко
- **На таких слайдах** изображение не обязательно (можно обойтись без картинки)

---

## Требования к викторинам

### Промежуточные викторины (10 блоков)
- **В каждом блоке:** ровно **3 вопроса**
- **Тип:** только `multiple-choice` (выбор из 4 вариантов)
- **Вопросы** должны проверять материал из **предыдущих 4 слайдов**
- **Объяснение:** обязательно для каждого вопроса, 1–2 предложения

### Формат вопроса
```
- Вопрос: короткий, ясный
- 4 варианта ответа (один правильный)
- Правильный ответ: индекс 0, 1, 2 или 3
- Объяснение: почему этот ответ верен
```

## Примеры хороших текстов слайдов

**Пример 1 (конкретика, экономика):**
```
Монгольский всадник вёл 3–5 лошадей. Одна для езды, остальные — запас еды.
В походе монголы пили кобылье молоко (кумыс) и ели конину. Одна лошадь = 200 кг мяса.
Это решало проблему провианта в степи.
```

**Пример 2 (скандал, цифры):**
```
В 388 году до н.э. спартанец Евпол подкупил трёх боксёров, чтобы они проиграли ему.
Скандал раскрылся. Элланодики оштрафовали всех четверых на 6 талантов каждого.
На эти деньги поставили шесть бронзовых статуй Зевса у входа в стадион — Заны. Это были монументы позора.
```

**Пример 3 (экономика, связь с последствиями):**
```
Александр потратил 180 талантов на похороны Гефестиона — больше, чем годовой доход Афин.
Это было политическое заявление: показать богатство и власть.
Но казна была пуста, и через год после смерти Александра империя обанкротилась.
```

**Пример 4 (Рим, быт):**
```
Инсула — многоквартирный доходный дом, в котором жило большинство римлян.
Не дворец и не вилла: тесные комнаты, сдаваемые внаём. В одном таком здании могли ютиться сотни людей.
```

**Пример 5 (анекдот/юмор):**
```
Диоген Синопский, киник, жил в бочке и презирал богатство.
Когда его спросили, почему философы берут деньги с учеников, а он — нет, он ответил:
«Потому что они считают, что их учение чего-то стоит. Я — нет.» Ирония над теми, кто торговал мудростью.
```

## Примеры плохих текстов (чего избегать)

| ❌ Плохо | ✅ Хорошо |
|----------|-----------|
| «Монголы были сильными» | «Монголы проходили 100–150 км в день» |
| «Благородные рыцари сражались за честь» | «Рыцари зарабатывали на выкупах пленных» |
| «Олимпиада была важным событием» | «В 388 году до н.э. спартанец Евпол подкупил трёх боксёров…» |
| «Древний Рим был великим» | «Большинство римлян жили в инсулах — многоквартирных трущобах» |

## Каталог тем

Темы выбираются из каталога `docs/TOPICS_CATALOG.md`. Каждая тема имеет:
- Название
- Подтемы и ключевые аспекты (экономика, конкретные факты, нетривиальные углы)

**Примеры тем:**
- Философия как бизнес: школы Афин — гонорары, конкуренция школ, богатство учеников
- Повседневная жизнь в Риме: инсулы и нищета — трущобы, цены, социальное неравенство
- Олимпийские игры: коррупция и допинг в античности
- Спартанцы: государственный террор
- Рыцарские турниры: кровь, деньги и политика

## Формат вывода для ИИ

При создании текстов ИИ должен предоставить:

### 1. Метаданные темы
```
id: theme-id (латиница, дефисы)
title: Название темы (русский)
description: Краткое описание 2–3 предложения
category: История / География / Наука / и т.д.
```

### 2. Структура занятия
```
structure: {
  greeting: 'Приветствие',
  mainContent: 'Описание основной части',
  interactive: 'Викторины',
  additional: 'Медиа',
  closing: 'Завершение'
}
```

### 3. Слайды (массив)
Для каждого информационного слайда:
```typescript
{
  id: 's1',
  title: 'Заголовок слайда',
  content: 'Текст 2–3 предложения.',
  duration: 3
}
```

Для слайдов с анекдотами — заголовок начинается с «Юмор: ».

### 4. Викторины (q1–q10)
Для каждого блока:
```typescript
{
  id: 'q1',
  title: 'Викторина',
  content: '',
  duration: 2,
  questions: [
    {
      id: 'q-theme-1-1',
      text: 'Текст вопроса?',
      type: 'multiple-choice',
      options: ['Вариант 1', 'Вариант 2', 'Вариант 3', 'Вариант 4'],
      correctAnswer: 1,  // индекс правильного (0–3)
      explanation: 'Объяснение правильного ответа.'
    },
    // ещё 2 вопроса
  ]
}
```

### 5. Итоговая викторина
30 вопросов — все вопросы из q1–q10, с id `final-1`, `final-2`, … `final-30`.

---

## Чек-лист перед сдачей

- [ ] Ровно 30 информационных слайдов + 10 викторин
- [ ] Каждый слайд 2–3 предложения, не более 150 слов
- [ ] 4–5 слайдов с анекдотами (заголовок «Юмор: …»)
- [ ] Нет повторений информации между слайдами
- [ ] Есть конкретика (цифры, даты, имена) там, где это нужно
- [ ] Цены — только с сопоставлением (эквивалент)
- [ ] Стиль: реалистичный, без романтизации
- [ ] В каждой викторине 3 вопроса по предыдущим 4 слайдам
- [ ] Все 30 вопросов итоговой викторины соответствуют промежуточным
