# Быстрый старт

## Установка

```bash
cd projects/Tagesförderstätte
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на `http://localhost:3000`

## Структура занятий

Занятия находятся в `src/data/sessions.ts`. Каждое занятие включает:

- **Метаданные**: название, описание, категория, тип, длительность
- **Слайды**: презентационные слайды с текстом и медиа
- **Викторина**: вопросы для закрепления материала
- **Медиа**: изображения, видео, аудио
- **Структура**: приветствие, основная часть, интерактив, завершение

## Добавление медиа

1. Поместите файлы в `public/media/` (создайте подпапки по темам)
2. Укажите путь в занятии: `/media/theme/file.jpg`

## Примеры занятий

В проекте уже есть два примера:
- "Римские императоры: от Августа до Нерона"
- "Берлин в 1920-е годы: Золотые двадцатые"

## Создание нового занятия

1. Откройте `src/data/sessions.ts`
2. Добавьте новый объект `Session`
3. Заполните все необходимые поля
4. Занятие появится на главной странице автоматически
