# Изображения для темы «Шуты при дворе» (`court-jesters`)

Слайды ссылаются на JPG в этой папке (`/media/court-jesters/…`). Сами файлы можно добавить вручную: ориентиры для поиска и имена перечислены в **IMAGE_LIST.md** и **IMAGE_LIST.html** (ссылки на Google Images по каждому имени файла).

Рекомендации:

- разрешение достаточное для полноэкранного показа (например от 1200 px по длинной стороне);
- лицензия на использование (общественное достояние, CC или собственная съёмка);
- после добавления файлов проверьте пути в `src/data/court-jesters-session.ts` (они уже совпадают с именами в списках).
